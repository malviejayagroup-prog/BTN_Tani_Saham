📤 Upload ke GitHub

Cara Upload Semua File ke GitHub Pages
Repository: malviejayagroup-prog · Gratis · Selesai dalam 10 menit
1
Download semua file ini ke HP/laptop
Dari halaman Claude, download semua file berikut ke satu folder yang sama. Beri nama folder: TaniSaham
🌐 index.html Portal ini (rename dari portal.html) UTAMA
🌿 baheulatani-tanisaham.html Website utama investasi
📋 panduan-form-sheets-tanisaham.html Panduan Google Form & Sheets
📱 konten-7hari-tanisaham.html 7 konten medsos
🗺️ tani-free-roadmap.html Roadmap gratis ke berbayar
💼 TaniSaham-Anggaran-2026.docx Dokumen anggaran
2
Rename file portal menjadi index.html
File ini (portal) harus diberi nama index.html agar otomatis terbuka saat orang mengunjungi link GitHub Anda. Ganti nama file sebelum upload.
3
Buka repository GitHub Anda
Buka browser, login ke GitHub, lalu buka repository:
github.com/malviejayagroup-prog/Tani-saham-btn
4
Upload semua file sekaligus
Di halaman repository, klik "Add file" → "Upload files". Drag & drop semua file dari folder TaniSaham sekaligus. Klik "Commit changes".
5
Selesai! Semua file online
Dalam 1–2 menit semua halaman bisa diakses di:
malviejayagroup-prog.github.io/Tani-saham-btn/
Contoh akses panduan form:
malviejayagroup-prog.github.io/Tani-saham-btn/panduan-form-sheets-tanisaham.html
